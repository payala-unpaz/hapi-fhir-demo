### 🚀 Instalación de HAPI FHIR con Docker
En este laboratorio aprenderás a desplegar un servidor de interoperabilidad clínica utilizando Docker.

### Paso 1/4. Verificación inicial

Antes de comenzar, asegúrate de que el motor de Docker está activo en tu terminal:

Copia y ejecuta: 

```bash
docker --version
```{{copy}}

Luego revisa si hay otros contenedores corriendo: 
```bash
docker --ps
```{{copy}}
